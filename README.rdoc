== README

Heimdaller is a Ruby on Rails app for keeping track of Pathfinder Roleplaying Game character sheets. It is named after Valli Heimdaller, a Gnome Fighter.

It is currently not very operational.

Runs on Ruby 2.0.0-p125, Rails 4.0.0

Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
